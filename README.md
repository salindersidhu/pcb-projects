## Overview

Repository of personal PCB and electronics designs created in KiCad. Includes schematics, board layouts, revisions, and supporting documentation for various hardware projects ranging from prototypes to finalized builds.

## Prerequisite Software

| Software       | Version   |
| :------------- | :-------- |
| Git            | 2.20.1+   |
| Kicad          | 9.0+      |

## Projects

### [Stereo Speaker Amp](/stereo-speaker-amp/)

- A compact full range stereo amplifier PCB based on the TDA2822M IC.

### [USB PD Trigger Module](/usb-pd-module/)

- A configurable USB-C Power Delivery trigger module.

### [RGB LED Dimmer](/rgb-led-dimmer/)

- A compact analog PWM dimmer for independently controlling RGB LED brightness using 555 timers and MOSFET switching.
