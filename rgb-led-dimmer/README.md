# RGB LED Dimmer

[![License: Apache](https://img.shields.io/badge/license-APACHE-brightgreen.svg?style=for-the-badge)](/LICENSE.md)

**Rev:** 1.0

A compact analog RGB LED dimmer PCB using independent PWM control for each color channel. The design uses three 555 timer circuits to generate adjustable PWM signals that drive MOSFET switches for smooth brightness control of RGB LEDs.

![RGB LED Dimmer Rotating View](3d/rgb_led_dimmer.gif)

## Features

- **Independent RGB control:** Three separate PWM dimmer channels
- **PWM Generator:** 555 timer based oscillator per color channel
- **Adjustable Brightness:** 50kΩ potentiometer per channel
- **MOSFET Output Stage:** Logic-level N-channel MOSFET switching
- **Supply Voltage:** 12V DC input
- **LED Compatibility:** Designed for standard 5mm RGB LEDs with current-limiting resistors
- **PCB:** Compact 2-layer layout with ground plane
- **Output Connector:** Dedicated RGB LED output header
- **Modular Power Input:** Designed to pair with external regulated power modules

## Design Overview

### PWM Generation

Each color channel uses a dedicated **555 timer configured as an adjustable PWM oscillator**. The duty cycle is controlled using a 50kΩ potentiometer with steering diodes that allow independent control of charge and discharge paths.

This configuration provides:

- Smooth brightness adjustment  
- Wide PWM duty cycle range  
- Consistent LED brightness control without changing LED current  

### Output Stage

Each PWM signal drives the gate of a **logic-level N-channel MOSFET** through a gate resistor.

The MOSFET operates as a **low-side switch**, allowing efficient PWM dimming of the LED channels.

Key characteristics:

- Low gate drive requirements
- Minimal heat generation at typical LED currents
- Suitable for multiple LEDs per channel

### Power Input

The board operates from a **12V DC supply**.

A local **bulk capacitor** stabilizes the supply rail and reduces switching noise generated by the PWM dimming circuits.

Typical supply sources include:

- External 12V power supplies
- USB-C PD trigger modules
- DC bench supplies

### LED Output

The RGB LED output header provides connections for:

- **Red channel**
- **Green channel**
- **Blue channel**
- **Common supply rail**

Each LED must include its own **current-limiting resistor** to ensure safe operation.

## Electrical Specifications

- **Input Voltage:** 12V DC  
- **PWM Channels:** 3 (Red, Green, Blue)  
- **Control Method:** Analog PWM dimming  
- **Typical Load:** Up to 4 standard RGB LEDs per channel (with resistors)  
- **Switching Devices:** Logic-level MOSFETs  

## Key Components

- **U1–U3:** 555 Timer ICs (PWM generators)  
- **Q1–Q3:** IRLML6344 N-channel MOSFETs (LED switching)  
- **VR1–VR3:** 50kΩ potentiometers (brightness control)  
- **D1–D6:** Signal diodes (PWM steering network)  
- **C10:** Bulk supply capacitor  
- **J1:** RGB LED output connector  
- **J2:** Power input connector
